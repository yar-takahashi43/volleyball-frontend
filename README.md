新規試合（スコアシート）

- 試合保存機能
- 登録してある選手を選べる
- 登録してある相手を選べる
- 登録してなくてもスコアシート上で登録もできる？？？

試合一覧

- 過去の試合を取得する（全件取得）
- 対戦相手で取得
- できれば日にち順に

選手登録

- 背番号とフルネームで登録する
- 表示する名前を決めることができる

選手一覧

- 選手を全件検索
- 名前か背番号で検索する
- 更新や削除ができる

対戦相手登録（スコアシートからも追加できる。）

- 一意の名前で登録する

対戦相手一覧

- 相手を全件検索
- 名前で検索する
- 更新や削除ができる
